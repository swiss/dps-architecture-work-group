Web Service ()
Definition: Web Service ABB is an Application Interface that enables interoperability between different applications or systems over the Internet Source: W3C Source reference: https://www.w3.org/TR/2004/NOTE-ws-arch-20040211/#relwwwrest Example: The following implementation is an example on how this specific Architecture Building Block (ABB) can be instantiated as a Solution Building Block (SBB): Web Services Description Language (WSDL) - WSDL is an XML format for describing network services as a set of endpoints operating on messages containing either document-oriented or procedure-oriented information. The operations and messages are described abstractly, and then bound to a concrete network protocol and message format to define an endpoint. https://www.w3.org/TR/wsdl.html Interoperability Dimension: Behavioral IoP LOST view: Technical view - application Interoperability Saliency: The Web Service Application Interface ABB is salient for interoperability because, from the beginning, Web services can be interoperable by design. By putting in place web services, interoperability can be guaranteed between systems and applications, and therefore, among digital public services.
