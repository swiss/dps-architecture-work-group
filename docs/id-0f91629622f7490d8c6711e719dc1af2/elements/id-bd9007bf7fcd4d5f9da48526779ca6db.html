Agent ()
Definition: Agent ABB is a Business Role defined as an Entity that is able to carry out action. Source: SEMIC-CPV or SEMIC-CCCEV Source reference: https://semiceu.github.io/Core-Person-Vocabulary/releases/2.00/#Agent Example: The following implementation is an example of how this specific Architecture Building Block (ABB) can be instantiated as a Solution Building Block (SBB): foaf:Agent An agent (eg. person, group, software or physical artifact).
