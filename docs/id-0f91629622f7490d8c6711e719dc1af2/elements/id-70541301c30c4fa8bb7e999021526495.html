DVS Short Term Coordination (2) ()
Short Term Coordination, Harmonisation & Infrastructure System 2 represents the information channels and bodies that allow the primary activities in System 1 to communicate between each other and which allow System 3 to monitor and co-ordinate the activities within System 1. Represents the scheduling function of shared resources to be used by System 1.
