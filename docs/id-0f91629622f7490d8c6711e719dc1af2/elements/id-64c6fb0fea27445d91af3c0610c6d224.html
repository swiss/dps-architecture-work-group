ech-0122 Eine natürliche oder juristische Person, welche eine Leistung einer Behörde in Anspruch nimmt (manche sprechen in diesem Zusammenhang von „Kunden“) oder aufgrund sanktionierter Rechte und Pflichten eine öffentliche Leistung einer Behörde beziehen kann bzw. beziehen muss. Im Rahmen der gesetzlichen Mitwirkungspflicht muss sie sich u.U. an der Leistungserstellung beteiligen. Ein Partner kann sein: - eine Behörde - ein externer Partner – eine natürliche oder juristische Person, Organisation oder ausländische Behörde, welche Leistungen von einer Schweizer Behörde bezieht. eCH-0219-2
