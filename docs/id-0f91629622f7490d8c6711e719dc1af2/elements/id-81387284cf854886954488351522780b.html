Geschäftsrolle ()
Eine Geschäftsrolle (Business Role) repräsentiert die Verantwortung für die Ausführung bestimmter Verhaltensweisen, die einem Akteur zugewiesen werden können, oder die Rolle, die ein Akteur in einer bestimmten Aktion oder einem Ereignis spielt.
