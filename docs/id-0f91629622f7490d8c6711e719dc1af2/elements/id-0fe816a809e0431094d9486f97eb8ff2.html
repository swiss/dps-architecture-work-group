Infrastrukturen ()
eCH-0279 Kap5.5 Die Infrastrukturen umfassen alle Hardware-nahen IKT-Elemente, die für die Umsetzung der oberen Ebenen notwendig sind. Sie ermöglichen die Speicherung (Speicher), den Austausch von Daten (Datenkommunikation) und die Durchführung von Berechnungen (Rechner). Sensoren ermöglichen, Informationen über die Realwelt zu erheben. Aktoren dienen der aktiven Beeinflussung der Realwelt.
