Org-ID «Kontexte» stellen ein organisatorisches Gebilde dar, in welches «Subjekte» eingegliedert werden. Diese Gebilde sind streng hierarchisch strukturiert. Den obersten Kontext einer institutionalisierten Stammdatenlieferung («Institutionskontext») stellt die juristische Person dar, welche die Stammdaten sendet. eCH-0291 Organisatorisches Gebilde, in welches Subjekte eingegliedert werden. Ein Kontext KANN ausrüstende (z.B. berechtigungssteuernde) Bedeutung haben.
