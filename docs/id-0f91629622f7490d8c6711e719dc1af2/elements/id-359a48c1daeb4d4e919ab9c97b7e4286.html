Treiber (intern oder extern) ()
Ein Treiber (Driver) repräsentiert eine externe oder interne Bedingung, die eine Organisation dazu motiviert, ihre Ziele zu definieren und die notwendigen Veränderungen zur Erreichung dieser Ziele umzusetzen.
