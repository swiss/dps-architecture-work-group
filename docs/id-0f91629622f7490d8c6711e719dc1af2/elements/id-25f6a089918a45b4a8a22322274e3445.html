Geschäftsfähigkeiten mit Voraussetzungscharakter ()
Geschäftsfähigkeiten mit Voraussetzungscharakter (siehe eCH-0122 ([6])) sind Fähigkeiten zum Verständnis des gemeinsamen Kontexts der Organisationen, soweit dieser für die Leistungserbringung notwendig ist («die Welt gemeinsam wahrzunehmen»). Hierzu gehören z. B. natürliche Personen, Unternehmen, Tiere, Gebäude, Geo-Informationen, Wetter oder Rechtssammlung. Sie umfassen die folgenden Themen (Zitat gem. eCH-0122):
