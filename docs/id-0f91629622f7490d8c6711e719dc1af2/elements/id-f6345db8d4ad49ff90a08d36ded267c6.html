TRiG ()
Description: TRiG is designed to express RDF data involving multiple named graphs or contexts. It allows the grouping of triples into different graphs, providing a way to differentiate and organize data within an RDF dataset. TRiG syntax is similar to Turtle, but it includes additional features for representing named graphs.
