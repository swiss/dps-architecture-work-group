Ausgabewert des Authentifikators ()
eCH-0107 Schnittstelle Wird durch eine mathematische Funktion (Authentifikator oder Authentifizierungsfunktion) aus einem geheimen Wert (z. B. privater Schlüssel), einem oder mehreren optionalen Aktivierungswerten (z. B. PIN oder biometrischer Informationen), und einem oder mehreren optionalen Eingabewerten (z. B. Zufallswerten oder Challenges) generiert. Der Ausgabewert des Authentifikators ist so stark wie das verwendete Verfahren bzw. dessen Implementation.
