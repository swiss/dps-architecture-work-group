Common Data Model ()
Description: CDM is an ontology based on the FRBR model (Functional Requirements for Bibliographic Records). CDM uses RDF(S)/OWL technologies, capable of representing relationships between the resource types managed by the Publications Office and their views. All within the scope of the FRBR model and, therefore, in terms of Work, Expression, Manifestation and Item.
