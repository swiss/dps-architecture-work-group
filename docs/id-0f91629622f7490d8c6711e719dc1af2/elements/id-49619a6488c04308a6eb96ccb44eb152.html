MKK04 - IKT-Trainings- & Kompetenzmanagement ()
IDie Fähigkeit, das Lernen zu managen und die Bereitstellung von Schulung, Ausbildung und Wissen für Mitarbeiter zu unterstützen, um sicherzustellen, dass sie über die richtigen Kompetenzen und Informationen verfügen, die sie benötigen. Dazu gehört auch die Mitarbeiterentwicklung durch Erstellung von Entwicklungsplänen sowie die Koordination, Durchführung und das Monitoring von Trainingsmassnahmen. Die Entwicklung von Mitarbeiterfähigkeiten kann z.B. von einer Digitalen Strategie abgeleitet werden (d.h. individueller Kompetenzaufbau durch Trainings)
