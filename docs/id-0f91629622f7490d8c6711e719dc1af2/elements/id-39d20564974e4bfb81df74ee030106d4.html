Containers Manager ()
Definition: Containers Manager ABB is a Technology Service that implements the process of organizing, adding or replacing large numbers of software containers Source: Inspired in referent cloud service providers Source reference: Synonym: Example: The following implementation is an example on how this specific Architecture Building Block (ABB) can be instantiated as a Solution Building Block (SBB): OpenSHIFT is a secure and an enterprise-grade cloud-based container orchestration system with Kubernetes at the backend.
