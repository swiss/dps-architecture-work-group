Geschäftsprozess ()
Ein Geschäftsprozess (Business Process) repräsentiert eine Abfolge von Geschäftsverhalten, die ein bestimmtes Ergebnis erzielt, wie beispielsweise eine definierte Menge von Produkten oder Geschäftsdiensten.
