Once-Only-Prinzip ()
Zitat aus eCH-0279, 5.2.2: Die öffentlichen Verwaltungen stellen sicher, dass betroffene Personen dieselben Informationen nur einmal an eine öffentliche Verwaltung übermitteln müssen. Öffentliche Verwaltungen ergreifen unter Berücksichtigung der geltenden Datenschutzbestimmungen Massnahmen, dass sie diese Daten wei-terverwenden dürfen, so dass keine zusätzliche Belastung für betroffene Personen und Stakeholder entsteht.
