eCH-0219-2 Ding Synonym Engl. Thing Ein Ding im IAM-Kontext ist ein physischer Gegenstand, welcher über ein Netzwerk erreichbar gemacht werden kann. Innerhalb des Netzwerkes ist das Ding mit einem Identifikator eindeutig identifizierbar. Mehrere Dinge, welche über ein Netzwerk verknüpft sind, bilden ein Internet der Dinge (Internet of Things, IoT). Dinge können weitere Dinge enthalten. Ein Ding gehört immer zu einer Organisation oder zu einer natürlichen Person. Synonyme: • Objekt • Engl. Digital Identity, Electronic Identity
