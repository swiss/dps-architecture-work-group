Hash Code ()
Definition: Hash Code ABB is a Data Object that enables the implementation of a numeric or alphanumeric string obtained by the application of an algorithm that turns a variable amount of data into a small, fixed-length value Source: Inspired in CROS Source reference: https://ec.europa.eu/eurostat/cros/content/data-hashing_en Example: The following implementation is an example of how this specific Architecture Building Block (ABB) can be instantiated as a Solution Building Block (SBB): The hash code is a relevant tool used, for instance, in the personal data pseudonymisation technique. https://edps.europa.eu/sites/edp/files/publication/19-10-30_aepd-edps_paper_hash_final_en.pdf Interoperability Dimension: Structural IoP LOST view: Semantic view Interoperability Saliency: The Hash Code Data Object ABB is salient for semantic interoperability because it supports semantic interoperability by providing alphanumeric strings comprising a variable amount of data. This enables and supports interoperability by reducing data and changing from comprehensible data to anonymised strings.
