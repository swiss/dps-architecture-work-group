Org-ID Kontexte müssen an Subjekte zugewiesen werden, damit sich ihre Wirkung im Empfänger entfalten kann. Da ein Subjekt in mehreren Kontexten kollaborieren kann und dies mit unterschiedlichen Gültigkeitsräumen und Rechtsnaturen, braucht es dafür eigenständige Entitäten. eCH-0291 Organisatorisches Gebilde, in welches Subjekte eingegliedert werden. Ein Kontext KANN ausrüstende (z.B. berechtigungssteuernde) Bedeutung haben.
