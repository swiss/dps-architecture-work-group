IVP02 - Prozessintegration ()
Die Fähigkeit von Prozessen über mehrere IT-Lieferanten in einer Partnerschaft hinweg zu integrieren. In einer Multiprovider-Umgebung müssen Prozesse über mehrere Parteien hinweg funktionieren. Hier bezieht sich diese Fähigkeit insbesondere auf IT-Servicemanagementprozesse: Bsp. IT-Changemanagement: Bei einer Änderung an einem integrierten Service wird die Verantwortung über alle an der Änderung beteiligten Partner hinweg wahrgenommen. Bsp. Incidentmanagement: Der Kunde muss nicht die einzelnen Prozesse der beteiligten IT-Lieferanten kennen und bedienen, sondern kann eine Störung an einer zentralen Stelle melden. Diese wird in einem integrierten Prozess behandelt.
