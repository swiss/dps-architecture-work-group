Standardmässig interoperabel ()
Zitat aus eCH-0279, 5.2.3: Öffentliche Dienste sind so gestaltet, dass sie in der ganzen Schweiz und über organisatori-sche Grenzen hinweg erbracht werden können. Dazu ist der freie Austausch von Daten und digitalen Dienstleistungen in der Schweiz und international gewährleistet.
