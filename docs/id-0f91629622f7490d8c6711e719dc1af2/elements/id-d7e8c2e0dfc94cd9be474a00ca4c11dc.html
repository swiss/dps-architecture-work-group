GML ()
Description: Geography Markup Language (GML) is an XML-based format that is used to encode geographic information, including the geometry and properties of geographic features. It is designed to facilitate the exchange of geographic data between different systems and applications, and is commonly used in domains such as geographic information systems (GIS), web mapping, and location-based services. The Open Geospatial Consortium (OGC) is responsible for maintaining GML, and the latest version of the format is GML 3.2.1, which was released in 2019.
