Fehlende Rechtsgrundlagen ()
Viele der bestehenden rechtlichen Grundlagen sind auf eine nicht digitale Welt ausgelegt und unterstützen durchgängige digitale Leistungen als Gesamtsystem unzureichend. Regulatorische Vorgaben erschweren teilweise eine nutzerfreundliche und in technischer Hinsicht wirtschaftliche Umsetzung von Digitalisierungsvorhaben. Fehlende rechtliche Grundlagen und datenschutzrechtliche Befürchtungen erschweren eine zukunftsgerichtete Wertschöpfung aus Daten zum Nutzen der Anspruchsgruppen.
