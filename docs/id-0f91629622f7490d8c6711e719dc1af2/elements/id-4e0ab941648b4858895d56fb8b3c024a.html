JSON-L ()
Description: JSON Lines, also known as newline-delimited JSON, is a text format designed for storing structured data that can be processed one record at a time. It provides a convenient and efficient way to store and exchange data in a format that is compatible with Unix-style text processing tools and shell pipelines.
