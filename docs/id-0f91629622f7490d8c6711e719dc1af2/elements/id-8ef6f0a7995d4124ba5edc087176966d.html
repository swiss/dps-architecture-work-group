EAB09 - Service Desk Management ()
Die Fähigkeit zur zeitnahen und effektiven Behandlung von Benutzeranfragen und Störungsmeldungen. Die Fähigkeit umfasst den Endusersupport über einen Service Desks (zentrale Anlaufstelle), die Behandlung von Benutzeranfragen (Service Request Management) sowie die Störungsbearbeitung (Incident Management). Dazu werden die benötigten Supportfunktionen (Service Desk, 1st-, 2nd-, und 3rd Level Support) organisiert und eingebunden. Dem Benutzer wird eine zentrale Anlaufstelle präsentiert ggf. auch über ein einheitliches Serviceportal (siehe Fähigkeit Serviceportalmanagement), worüber Self-Help und Self-Service Funktionalitäten zur Verfügung stehen.
