Führungsfähigkeiten ()
eCH-0122 Geschäftsfähigkeiten zur Führung Dieser Bereich umfasst das interne Management jeder einzelnen Organisation (z.B. Bundesverwaltung, Kanton, Gemeinde), aber auch das Management des Gesamtsystems der Behörden der Schweiz. Für Letzteres werden übergreifende Führungsinstrumente benötigt, damit zur E-Government-Strategie konforme Entscheide getroffen werden und die Umsetzung konsistent und nutzbringend durchgeführt wird.
