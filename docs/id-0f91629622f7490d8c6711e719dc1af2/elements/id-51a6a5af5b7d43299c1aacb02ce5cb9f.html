Geschäftsdienst ()
Ein Geschäftsdienst (Business Service) repräsentiert explizit definiertes Verhalten, das von einer Geschäftsrolle, einem Geschäftsakteur oder einer Geschäftskollaboration in ihrer Umgebung bereitgestellt wird.
