Entität IAM ()
eCH-0219-2 Entität Synonym: Entity Eine Entität ist ein Element eines IT-Systems, z. B. eine Komponente oder ein Teilsystem, welche als digitale Identität in einem digitalen Prozess handelt (siehe Abbildung 7 - Objekt und Subjekt). Beispiele für Entitäten sind: • Identity Provider, • RPs, • Services, • Applikationen, • Autonome Bots.
