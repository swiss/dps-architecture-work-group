RCR07 - Cyber Monitoring und Alerting ()
Die Fähigkeit der Überwachung aller IKT-Systeme und Betriebsmittel sowie deren Nutzung, der Überwachung von Aktivitäten aller Parteien, das Aufspüren von Schadsoft-ware sowie die Durchführung von Verwundbarkeitsscans. Alle Systeme und Betriebsmittel werden laufend überwacht, um sicherzustellen, dass Aktivitäten/Zugriffe von unberechtigten Personen, Geräten und Software erkannt werden. Dies ermöglicht die Entdeckung von Cybersecurity-Vorfällen sowie die Über-prüfung der Effektivität der Schutzmassnahmen. Dazu gehört es ein kontinuierliches Netzwerkmonitoring, ein Monitoring aller physischen Betriebsmittel und Gebäude, der IKT-Nutzung der Mitarbeitenden, sowie der Aktivitäten von externen Dienstleistern zu etablieren. Es muss ausserdem sichergestellt werden, dass Schadsoftware, auch auf Mobilgeräten, entdeckt werden kann und, dass regelmässig Verwundbarkeitsscans durchgeführt werden.
